# 🎨 Painting Shop Website

A simple **Painting Shop Website** built using **HTML, CSS, and JavaScript**.  
This website allows users to explore different paintings, search for paintings, and view products in a simple shopping interface.

## 🌐 Live Demo
https://painting-shop-project.vercel.app/

---

## 📌 Features
- Display different paintings
- Search paintings using search bar
- Add paintings to cart
- Simple and clean user interface
- Responsive design

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Vercel (for deployment)

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/Hetvi4728/painting-shop-project.git
```

2. Open the project folder.

3. Run the project by opening **index.html** in your browser.
